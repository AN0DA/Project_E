Temperature is generated based on previously generated neightbours and amplitude percent set in env_temperature.h. In first place generated is block [0, 0], then first line and after that first column. Rest of array use average value from blocks [n-1, m], [n, m-1] and [n-1, m-1].

Temperature mixing is based on making average value from adjacent fields, assuming SE wind. First mixed is first row with [0][i-1] field, then first column with [i-1][0], then others with [i-1][j], [i][j-1] and [i-1][j-1].
